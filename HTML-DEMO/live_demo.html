Hierarchical Policy Decomposition
High-Level Policy (Manager) πH

Monitors global grid exploration state, battery status of all agents, and coordinates victim locations. Every macro-interval, it selects a strategic sector quadrant (NW, NE, SW, SE) or assigns specific scanned victims/debris to agents. Trained using tabular Q-learning under the SMDP framework.

Low-Level Policy (Worker) πL

Directly interacts with the grid. Given the designated target coordinates assigned by the Manager, the Worker observes its local surroundings (boundaries, static debris, dynamic fire) and its battery, outputting directional steps (North, South, East, West) or interact actions. Optimized via Deep Q-Network (DQN) with PyTorch.

Explainability Module (XAI)

Computes Q-value differentials to explain macro goal assignments and computes backpropagated Q-value gradients (saliency maps) with respect to input features (target direction, battery, obstacles) to explain step-by-step navigation actions in real-time.

Policy Equations
SMDP Options Update
Q(s, g) ← Q(s, g) + α [ R + γτ max Q(s', g') - Q(s, g) ]
R = Option cumulative discount reward, τ = Option step duration
Low-Level Bellman Error
L(θ) = E [ ( r + γ max Q(s', a'; θ-) - Q(s, a; θ) )2 ]
Saliency Gradient attribution
Saliency(f) = | ∂Q(s, a) / ∂sf |
Attribution score for state feature f indicating decision influence

Cooperative Coordination Strategy

The EMARL-SAR framework addresses path-blocking debris and fire hazard constraints through cooperative behaviors:

  • Parallel Reconnaissance: Drones split grid quadrants NW/NE/SW/SE to scan regions in parallel, avoiding coordinate overlap.
  • Heterogeneous Path-Clearing: Scout Drones detect debris path blockages and transmit targets to Rover A3, which navigates to clear path blockages.
  • Dynamic Fire Adaptation: Spreading fires block cells dynamically, forcing agents to run real-time local target re-routing.
